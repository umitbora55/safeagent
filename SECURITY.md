# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SafeAgent, please report it responsibly.

**Email:** safeagent.security@proton.me
**GitHub:** [Security Advisories](https://github.com/umitbora55/safeagent/security/advisories/new)

**Do not** open public GitHub issues for security vulnerabilities.

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Fix target:** Within 7 days for critical issues

## Scope

- Vault encryption bypass
- Prompt injection guard bypass
- Credential exposure in logs or output
- Dependency vulnerabilities

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
